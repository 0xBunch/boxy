## Workflows

### Daily (5-10 min)
1. Quick triage of new Flow items
2. Energy-score and lens-match
3. Extract Sparks from notable items
4. Note one "hottest" thing

### Weekly (30-45 min)
1. Full Flow review (clear the backlog)
2. Lens health check
3. Idea pipeline review
4. Pick 1-2 Ideas to advance
5. Archive stale items

### Monthly (60 min)
1. Output performance analysis
2. Source quality audit
3. Lens evolution review
4. Pattern retrospective
5. Next month focus

### Shipping Sprint
1. Pick one Idea
2. Go to ğŸš€ Ship Mode
3. Stay focused until done
4. Create Output record
5. Capture learnings
6. Celebrate
